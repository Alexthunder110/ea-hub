EA Trading Hub
Einmalige Einrichtung — dauert ca. 3 Minuten
1. Öffne sheets.google.com und erstelle ein neues, leeres Spreadsheet.
2. Klicke auf Erweiterungen → Apps Script
3. Lösche den vorhandenen Code, füge den Inhalt der Datei apps-script.gs ein und speichere (Strg+S).
4. Klicke auf Bereitstellen → Neue Bereitstellung → Typ: Web-App → Zugriff: Jeder → Bereitstellen.
5. Kopiere die Web-App-URL und füge sie hier ein:
⚠ Offline — Änderungen werden lokal gespeichert und beim nächsten Start synchronisiert.
